# Forbes Data Analysis Project

Проект для анализа данных миллиардеров из рейтинга Forbes с использованием Java и SQLite.

## Описание проекта

Проект выполняет следующие задачи:
1. Парсинг данных из CSV-файла Forbes
2. Нормализация данных и сохранение в SQLite БД
3. Выполнение аналитических SQL-запросов
4. Визуализация результатов в виде диаграмм

## SQL-запросы

Проект выполняет три запроса согласно варианту задания:

1. **Общий капитал по странам** - показывает топ-10 стран по общему капиталу миллиардеров
2. **Самый молодой миллиардер из Франции** с капиталом > 10 млрд долларов
3. **Бизнесмен из США** с самым большим капиталом в сфере Energy

База данных
Проект создает базу данных forbes.db с тремя нормализованными таблицами:

Таблица countries

CREATE TABLE countries (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT UNIQUE NOT NULL
);
Таблица industries

CREATE TABLE industries (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT UNIQUE NOT NULL
);
Таблица billionaires

CREATE TABLE billionaires (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    rank INTEGER NOT NULL,
    name TEXT NOT NULL,
    net_worth REAL NOT NULL,
    age INTEGER,
    country_id INTEGER,
    source TEXT,
    industry_id INTEGER,
    FOREIGN KEY (country_id) REFERENCES countries(id),
    FOREIGN KEY (industry_id) REFERENCES industries(id)
);
